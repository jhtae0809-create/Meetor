# MEETOR (미토) - AI 기반 진로탐색 멘토링 플랫폼

## 1. Project Overview (프로젝트 개요)
MEETOR는 "무엇을 하고 싶은지 모르는" 초·중·고 학생들에게 다양한 분야의 현업 전문가를 연결해주는 AI 기반 진로탐색 플랫폼입니다.
입시 중심의 단일 트랙에서 벗어나, 학생들이 동경하는 유튜버, 운동선수, 디자이너, 셰프, 기술 장인 등 실제 실무자와 1:1 실시간 화상 면담을 진행할 수 있도록 지원합니다. 

단순한 매칭을 넘어, AI 강점 진단 및 적성 검사 분석을 통해 학생에게 가장 적합한 멘토를 추천하고, 세션 종료 후 '진로 탐색 보고서'를 산출물로 제공하는 것을 목표로 합니다.

## 2. CareerNet Open API Utilization (커리어넷 공공데이터 활용 계획)
본 프로젝트는 교육부 산하 커리어넷의 진로·직업 데이터를 핵심 기반으로 사용하여 매칭 엔진의 신뢰도를 확보합니다.

* **활용 API:** 진로심리검사(직업적성·직업흥미·직업가치관 검사), 직업백과, 학과정보 API
* **활용 목적:**
  1. 학생이 입력한 자기소개서 및 성향 데이터를 커리어넷의 표준화된 직업/학과 분류 체계와 결합하여 정규화.
  2. LLM(대규모 언어 모델)을 활용한 텍스트 분석 시, 커리어넷의 공신력 있는 직업 적성 데이터를 추천 알고리즘의 표준 축으로 사용하여 학부모와 공교육 기관이 신뢰할 수 있는 매칭 결과 도출.

## 3. Core Features (핵심 기능)
* **AI 멘토 매칭 엔진:** LLM 기반 서술형 설문 분석 및 멘토 프로필 임베딩 유사도 계산을 통한 자동 추천.
* **실시간 1:1 화상 면담:** WebRTC 기반의 온라인 멘토링 환경 제공 (전 세션 녹화 및 보호자 참관 기능 포함).
* **진로 탐색 리포트:** 세션 종료 후 생활기록부 진로활동란 연계 및 학교 제출이 가능한 산출물 자동 발급.
* **미성년자 보호 설계:** 멘토 성범죄 경력 조회, 법정대리인 동의, 플랫폼 내 통신 의무화를 통한 안전한 환경 구축.

## 4. Tech Stack (기술 스택)
* **Frontend:** Responsive Web, WebRTC (화상 통신) 
* **Backend & AI:** Python, LLM API (GPT-4/Claude/Gemini), Text Embedding Models 
* **External Integration:** CareerNet Open API, Payment Gateway (PG)
